Usage Guidelines
Brand Colors
  • Primary Dark (#23416C): Use for all headings, important text, and primary brand elements
  • Primary Light (#457b9d): Use in gradients with Primary Dark, accent elements
  • Success Green (#10b981): Only for checkmarks and positive indicators
  • Error Red (#ef4444): Only for X marks and items to exclude
Typography Hierarchy
  • Section Titles (1.8rem): Major document sections
  • Phase/Card Titles (1.3rem): Individual phases, cards, major subsections
  • Subsection Titles (1.2rem): Within cards, methodology areas
  • Body Text (1.1rem): All paragraph content
Layout Rules
  • Maximum container width: 1200px
  • Use 2rem padding for main containers
  • Use 2rem gaps for grid layouts
  • All cards should have 1.5rem internal padding
  • Use border-radius: 8px for cards, 6px for smaller elements
  • Standard box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1)
Placeholder Text Conventions
  • [BRACKETS]: Content to be replaced with client-specific information
  • [CLIENT NAME]: Client organization name
  • [PROJECT TITLE]: Specific project name
  • [AMOUNT], [HOURS], [RATE]: Financial figures
  • [Activity N], [Objective N]: Numbered items in lists
